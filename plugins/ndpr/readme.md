### Introduction

NDPReforced (NDPR) is an cross server blocking plugin [NDP]（ https://github.com/No-Danger-Player-Project/NDP-MCDR ）The sequel aims to synchronize the ban lists between multiple servers in real-time, preventing malicious players from accessing any connected servers. By centrally managing player ban data, the security and management efficiency of server networks are significantly improved.
