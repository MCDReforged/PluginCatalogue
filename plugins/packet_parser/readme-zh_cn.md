# Packet Parser

用于解析 Minecraft JE 数据包的插件.

本插件与 [Login Proxy](https://github.com/kmcsr/login_proxy_mcdr) 配合使用.

辅助 LoginProxy 以及大部分依赖于 LP 的插件解析 Minecraft 数据包
