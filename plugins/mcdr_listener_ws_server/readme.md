### Introduction

🌐💬📡 WebSocket server for group-server bridge: forward player chat/join/leave events to chat platforms, receive text and images with in-game rendering, support remote command execution via RCON. 🎮🔗

- 🔑 Token authentication for secure WebSocket connections
- 🖼️ Image host whitelist for safe image rendering

For more details, refer to [README](https://github.com/VincentZyuApps/mcdr_listener_ws_server/blob/main/README.en-us.md)
